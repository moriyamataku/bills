# bills
請求書の作成及びPDFプレビュー

# Dependency
- Ruby (2.4.0)
- Rails (5.1.6)
- jQuery (1.12.4)

# Setup
```bash
bundle exec rake db:migrate
bundle exec rake db:seed
```

# Usage
`Sing up` して請求書情報を登録。
